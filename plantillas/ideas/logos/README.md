# Carpeta de Logos - IDEAS

Coloca aquí el logo de la revista IDEAS.

**Archivo requerido:**
- `logo_ideas.png` - Logo principal (formato PNG, recomendado 300 DPI)

**Dimensiones recomendadas:**
- Ancho: 600-1200 px
- Fondo transparente preferiblemente

**Formatos soportados:**
- PNG (recomendado)
- JPG
- PDF

Si no tienes un logo, el worker usará solo texto.
