# Carpeta de Archivos Subidos

Esta carpeta contiene los archivos originales subidos por los usuarios.

**Estructura:**
- `job-id.docx` - Documento Word subido
- `job-id.tex` - Documento LaTeX subido

**Importante:**
- Los archivos se eliminan automáticamente después de 30 días
- NO subir esta carpeta a Git (está en .gitignore)
- Asegurar permisos 755 para la carpeta

**Seguridad:**
- No permitir acceso directo vía web
- Configurar .htaccess para denegar acceso
